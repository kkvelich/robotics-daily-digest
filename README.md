# Robotics Market Sensing

Daily market-intelligence digest for edge AI in robotics. Auto-generated by a Claude Code remote agent every morning ~7 AM Phoenix.

**Read it as a website**: https://kkvelich.github.io/robotics-daily-digest/

## What's in each digest

1. Funding & M&A (>$10M robotics rounds, acquisitions, IPO filings)
2. Product launches & demos (humanoids, AMRs, cobots, dev kits)
3. Foundation models & robotics software (VLAs, world models, ROS / Isaac / OpenVINO / LeRobot)
4. Customer deployments (production rollouts at named customers)
5. Competitive silicon watch (NVIDIA, Qualcomm, Ambarella, Hailo, Rockchip, MediaTek)
6. China robotics ecosystem (humanoids, cobots, compute, policy, deployments)
7. Policy / standards / safety
8. Conferences & signals

Each digest cites every claim and skips anything older than 24 hours.

## Structure

- `_posts/YYYY-MM-DD-robotics-digest.md` — each day's digest, written by the agent
- `index.md` — auto-generated list of all digests
- `_config.yml` — Jekyll config (Cayman theme)
